# 1.1 NTP 시간 동기화란?

NTP(Network Time Protocol)는 네트워크의 모든 디바이스에서 시간을 동기화하는 데 사용되는 프로토콜입니다. 기본적으로 UDP의 123번 포트를 사용합니다.
